GTK3 Timer written in C

## Dependencies 
GTK3

## Usage
Takes in a time as a command line argument and starts a timer.  
Time can be formatted as SS, MM:SS, HH:MM:SS. Each field is not limited so SSSSSS or MMM:SSS is also permitted.
